# Metadata Group

Status: Ready for Implementation Planning
Version: MVP-001

## Purpose

Display label/value pairs for tea, teaware, and sessions.

## Layout

- Display: grid or flex column.
- Label to value gap: 4px.
- Group gap: 12px.
- Desktop metadata grid min column width: 160px.

## Typography

- Label: `type.caption`, `text.muted`.
- Value: `type.body-sm`, `text.primary`.

## Rules

- Units appear with values.
- Empty optional values may be hidden unless absence is meaningful.
