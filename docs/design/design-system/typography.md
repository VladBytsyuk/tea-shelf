# Typography

Status: Draft  
Version: MVP-001

## Goals

Typography must support calm scanning, bilingual branding, tea taxonomy, timer readability, and responsive layouts.

## Requirements

- Use a clear hierarchy for page titles, section headings, item names, metadata, and helper text.
- Timer numbers must be highly legible on mobile and desktop.
- Metadata such as tea type, region, material, and volume must remain readable in dense lists.
- Typography must support English and Russian product naming.
- Do not scale font size directly with viewport width.
- Avoid negative letter spacing.

## Usage Guidance

- Reserve large display type for primary screens or true hero moments.
- Use compact, readable headings inside catalog lists and timer panels.
- Keep form labels direct and practical.
- Long Russian or English labels must wrap without overlapping controls.
- Timer numerals must remain stable enough that countdown changes do not shift nearby controls.
