# Quality Gates

Status: Approved for Implementation
Version: MVP-001

## Spec Gates

- MVP PRD approved.
- Functional spec approved.
- Feature specs approved.
- UX flows approved.
- Core entity definitions approved.
- Acceptance criteria mapped to FR/NFR IDs.
- Risk register reviewed.

## Implementation Gates

- Unit or component checks pass for core logic and UI controls.
- End-to-end or equivalent flow checks pass for first-use, tea CRUD, teaware CRUD, custom session, tea session, session save, repeat session, archive/restore, and stock prompt.
- Data integrity checks pass for non-negative stock and session snapshots.
- Accessibility checks pass for keyboard, focus, labels, contrast, dialogs, and timer state.
- Responsive checks pass for mobile, tablet, and desktop.

## Release Gates

- Release build passes smoke validation.
- No open critical or high-severity defects.
- Medium defects are reviewed and explicitly accepted or fixed.
- Release evidence is attached to the release validation record.
- Deferred features are not required for MVP acceptance.
