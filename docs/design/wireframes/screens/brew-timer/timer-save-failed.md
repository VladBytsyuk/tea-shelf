# Timer Save Failed Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Let the user retry saving a completed session without losing session context.

## Layout

- Inline alert on completed-session screen, not a navigation away.
- Content: concise error title, retry action, and statement that session details remain available.

## Preserved Context

- Tea/custom label.
- Optional teaware.
- Sequence.
- Timestamps when available.
- Notes.
