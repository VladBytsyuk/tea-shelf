# Cancel Active Session Confirmation Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Confirm before discarding unsaved active-session progress or completed-session notes.

## Used By

- Active timer cancel action.
- Completed unsaved session cancel action.

## Layout

- Statement that unsaved progress or notes will be discarded.
- Destructive confirm action.
- Continue session action.

## Focus

- Trap focus while open.
- Restore focus to triggering control on close.

- AC-BT-006
