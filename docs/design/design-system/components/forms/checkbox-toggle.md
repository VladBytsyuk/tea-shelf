# Checkbox / Toggle

Status: Approved for Implementation
Version: MVP-001

## Purpose

Capture binary settings such as favorite filter or optional flags.

## Layout

- Row height: minimum 40px.
- Control size: 20px.
- Label gap: 8px.
- Label text uses `type.body`.

## Rules

- Checkbox is preferred for form inclusion.
- Toggle is allowed only for immediate mode/state changes where on/off semantics are clear.
- Favorite itself uses the favorite icon button, not a generic checkbox.
