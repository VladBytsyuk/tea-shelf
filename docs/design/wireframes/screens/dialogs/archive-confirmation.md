# Archive Confirmation Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Confirm archive actions before active records or sessions are hidden from default lists.

## Used By

- Tea archive.
- Teaware archive.
- Session archive.

## Layout

- Item/session name.
- Consequence summary.
- Destructive confirm action.
- Cancel action.

## Focus

- Trap focus while open.
- Restore focus to triggering control on close.
