# Prioritization Framework

Status: Draft  
Version: MVP-001

## Priority Classes

- MVP: Required for cataloging tea, cataloging teaware, or conducting guided brew sessions.
- MVP+: Valuable soon after MVP, but not required for first release.
- Deferred: Future roadmap work that must not enter MVP without change approval.

## MVP Must-Haves

- Tea CRUD.
- Teaware CRUD.
- Remaining stock tracking.
- Favorites.
- Basic filtering.
- Timer.
- Multi-steep sequence.
- Session save and history.
- Repeat saved session.
- Archive and restore records.
- Minimal custom session.

## MVP Should-Haves

- Basic teaware compatibility metadata.

## MVP Could-Haves

- Import/export.
- Advanced tasting notes.
- Beginner guidance prompts.

## Won't Have In MVP

- AI assistant.
- Articles and academy content.
- Payments and subscriptions.
- Marketplace.
- Social/community features.
- Native mobile apps.
- Cloud sync unless explicitly approved as MVP+.
- Standalone recipe favorites and recipe-management UI.
