# Textarea

Status: Ready for Implementation Planning
Version: MVP-001

## Purpose

Capture tea notes, teaware notes, and session notes.

## Layout

- Minimum height: 96px.
- Padding: 10px 12px.
- Border: 1px solid `border.default`.
- Radius: 6px.
- Width: 100%.
- Resize: vertical on desktop; fixed height or controlled auto-grow on mobile.

## Typography

- Value: `type.body`.
- Helper/error: field group rules.

## Rules

- Empty notes are valid.
- Notes are private user data.
- Save failure preserves text.
