# External Systems

Status: Draft  
Version: MVP-001

## MVP Position

No external systems are required for MVP.

## Explicitly Deferred External Systems

- Authentication provider.
- Cloud database or sync service.
- AI model provider.
- Content management system.
- Payment or subscription provider.
- Marketplace or affiliate integrations.
- Push notification service.

## Review Rule

Any external system added to MVP requires Product Manager approval, security/privacy review, architecture review, and traceability updates.
