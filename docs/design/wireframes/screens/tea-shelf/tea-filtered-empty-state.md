# Tea Filtered Empty State Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Show that the current tea filters hide all active teas.

## Layout

- Same content area as the list results.
- State that filters hide all active teas.
- Show active filter summary.
- Provide "Clear filters".

## Exit

- Clear filters returns to the active tea list or first-use empty state if no teas exist.
