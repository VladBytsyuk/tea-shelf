# Color System

Status: Draft  
Version: MVP-001

## Visual Direction

Chinese Modern Minimalism: calm, ritualistic, premium, accessible, and restrained.

## Palette Requirements

- Jade green: primary calm action and collection identity.
- Dark tea brown: text emphasis and grounding surfaces.
- Parchment beige: warm background and paper-like surfaces.
- Bamboo accent: secondary highlights.
- Deep red detail: sparse emphasis, warnings, and ceremonial accents.

## Semantic Requirements

- Primary action color.
- Secondary action color.
- Background colors.
- Surface colors.
- Text colors.
- Border colors.
- Success, warning, error, and unavailable states.
- Archived state.
- Timer running, paused, and complete states.
- Focus ring color.

## Accessibility Requirements

- Text and interactive states must meet contrast requirements.
- Timer and availability states must not rely on color alone.
- Deep red must be used sparingly and remain legible.
- Color tokens must have semantic names before implementation.

## Anti-Patterns

- Do not create a one-note palette dominated by one hue.
- Do not use decorative gradients as the primary visual identity.
- Do not let premium aesthetics reduce readability or tap clarity.
