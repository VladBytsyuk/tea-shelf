# Scalability

Status: Draft  
Version: MVP-001

## MVP Position

MVP scalability focuses on a single user's local collection and session history. Cloud scale is deferred until authentication and sync are approved.

## Requirements

- UI must remain usable with typical enthusiast collections.
- Lists must support filtering without layout instability.
- Timer behavior must not degrade under normal collection sizes.

## Future Considerations

- Account-backed storage.
- Cloud sync.
- Search/indexing for large collections.
- AI recommendation workloads.
