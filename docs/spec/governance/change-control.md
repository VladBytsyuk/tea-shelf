# Change Control

Status: Approved for Implementation
Version: MVP-001

## Change Request Requirements

- Problem or opportunity.
- Affected requirements and specs.
- MVP/MVP+/Deferred classification.
- Acceptance criteria impact.
- QA and release impact.
- Risk impact.
- PM decision.

## Automatic Escalation

Escalate changes involving AI, auth, cloud sync, external integrations, telemetry, permanent deletion, data export, payments, marketplace, native mobile, or social features.
