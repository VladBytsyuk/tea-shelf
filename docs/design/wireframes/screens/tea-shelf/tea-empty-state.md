# Tea Empty State Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Give first-time users a direct path to create their first tea.

## Layout

- Centered content inside the list body, not a full-page marketing screen.
- Content: short statement that no teas exist and one primary "Add tea" action.
- Exclusions: no AI prompt, marketplace link, article content, or expert taxonomy education.

## Exit

- Primary action opens Tea Form in create mode.
