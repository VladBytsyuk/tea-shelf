# MVP Pain Point Analysis

Status: Draft  
Version: MVP-001

## Pain Point 1: Users Forget Owned Teas

Problem: Tea enthusiasts may own many teas across types, brands, regions, and years, making memory unreliable.

MVP response:

- Tea Shelf inventory.
- Remaining quantity.
- Availability status.
- Favorites.
- Type and origin metadata.

Success criteria:

- User can answer what teas are currently available.
- User can find a favorite or available tea quickly.

## Pain Point 2: Users Do Not Know How To Brew Specific Tea

Problem: Brewing parameters vary by tea, teaware, and ceremony style.

MVP response:

- Multi-steep brew sequences.
- Temperature notes.
- Session history for repeatable practice.
- Repeat saved sessions.

Deferred response:

- AI recommendations.
- Knowledge Shelf articles.
- Beginner academy.

Success criteria:

- User can repeat a successful prior session.
- User can record useful timing and temperature notes.

## Pain Point 3: Users Do Not Track Steeping Times

Problem: Multi-steep ceremonies require attention and timing discipline.

MVP response:

- Brew Timer.
- Current steep display.
- Multi-steep advancement.
- Pause/resume/cancel states.
- Session completion record.

Success criteria:

- User can complete a ceremony using only Tea Shelf timer functionality.

## Pain Point 4: Users Do Not Know Suitable Teaware

Problem: Teaware material, volume, purpose, and brewing method affect the session.

MVP response:

- Teaware Shelf.
- Material and volume fields.
- Purpose and supported brewing methods.
- Optional teaware selection in sessions.

Deferred response:

- AI teaware recommendations.
- Advanced compatibility scoring.

Success criteria:

- User can identify practical teaware options before starting a session.

## Pain Point 5: Beginners Do Not Understand Chinese Tea

Problem: Beginners need learning support, but full education content would expand MVP.

MVP response:

- Clear fields.
- Gentle empty states.
- Simple first-use path.

Deferred response:

- Knowledge Shelf.
- Beginner academy.
- Guided recommendations.

Success criteria:

- Beginner can add first tea, add first teaware item, and complete first session without academy content.
