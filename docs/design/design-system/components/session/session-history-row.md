# Session History Row

Status: Approved for Implementation
Version: MVP-001

## Purpose

Summarize one saved session.

## Layout

- Base card layout: border, radius 8px, padding 16px.
- Mobile: stacked content and actions.
- Tablet/desktop: grid with identity, metadata, and actions.
- Minimum row height: 88px.

## Content Order

1. Tea name or custom session label.
2. Date/time.
3. Optional teaware snapshot.
4. Steep count.
5. Repeat action.
6. Archive action.

## Rules

- Uses saved snapshots only.
- Source record changes do not alter displayed values.
