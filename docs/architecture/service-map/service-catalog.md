## MVP Modules

| Module | Responsibility | Owner |
| --- | --- | --- |
| Tea Shelf | Tea inventory, favorites, stock, archive/restore | Engineering |
| Teaware Shelf | Teaware inventory, category/material/volume/method metadata, archive/restore | Engineering |
| Brew Timer | Timer setup, timer lifecycle, custom sessions, stock prompt | Engineering |

| Local Persistence | Store and retrieve MVP data locally | Engineering / Architect |

## Non-Services

AI, Knowledge Shelf, auth, cloud sync, commerce, and social systems are not MVP modules.
