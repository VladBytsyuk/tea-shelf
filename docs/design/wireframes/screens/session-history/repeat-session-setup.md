# Repeat Session Setup Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Start a new timer setup from saved snapshots without mutating the original session.

## Layout

- Timer Setup with source banner "Repeated from saved session".
- Content: copied tea/custom label, copied optional teaware snapshot, copied sequence, editable sequence before start.
- Archived sources: repeat does not require restoring source tea or teaware; unavailable source status is stated in the source banner when relevant.
