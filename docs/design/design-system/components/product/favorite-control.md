# Favorite Control

Status: Approved for Implementation
Version: MVP-001

## Purpose

Mark tea as favorite.

## Layout

- Icon button default size: 40px.
- Icon: 20px.
- Selected state uses `surface.selected`.

## States

- Unselected: outline/empty icon.
- Selected: filled/active icon plus selected state.
- Focus: global focus ring.

## Accessibility

- Accessible name changes between "Mark as favorite" and "Remove favorite".
- Exposes pressed state.
