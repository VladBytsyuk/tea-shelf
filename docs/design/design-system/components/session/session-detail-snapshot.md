# Session Detail Snapshot

Status: Approved for Implementation
Version: MVP-001

## Purpose

Show complete saved session context.

## Layout

- Reading max width: 760px.
- Section gap: 24px.
- Metadata grid min column width: 160px.
- Sequence snapshot uses Steep List display style.

## Sections

1. Session identity.
2. Tea snapshot or custom label.
3. Optional teaware snapshot.
4. Sequence snapshot.
5. Notes.
6. Actions.

## Rules

- Notes section shows empty state if no notes exist.
- Archived source records do not change snapshot display.
