# Spec Lifecycle

Status: Draft  
Version: MVP-001

## States

1. Draft
2. Cross-functional review
3. Revision
4. Approved for planning
5. Approved for implementation
6. Change controlled
7. Completed or archived

## Rules

- Implementation cannot start from conversation context alone.
- Approved specs must have acceptance criteria and traceability.
- Scope changes after approval require change-control updates.
