# System Overview

Status: Draft  
Version: MVP-001

## Purpose

Tea Shelf MVP is a responsive web application for managing personal tea inventory, teaware inventory, brew sessions, and session history.

## MVP Domains

- Tea Shelf: tea inventory, favorites, stock, active/archived lifecycle.
- Teaware Shelf: teaware inventory, brewing method metadata, active/archived lifecycle.
- Brew Timer: custom and tea-based multi-steep sessions.
- Session History: saved session snapshots, repeat session, archive/restore.

## Excluded Systems

- Authentication and cloud sync.
- AI assistant.
- Knowledge Shelf/content management.
- Commerce, marketplace, subscriptions, and social features.
- Native mobile applications.

## Implementation Boundary

The MVP may be implemented as a single web app, but internal modules must remain traceable to the domains above.
