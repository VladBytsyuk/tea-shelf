# Restore Confirmation Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Confirm restore actions when implementation requires an explicit restore check.

## Used By

- Archived tea restore.
- Archived teaware restore.
- Archived session restore.

## Layout

- Item/session name.
- Restore consequence.
- Confirm restore action.
- Cancel action.

## Focus

- Trap focus while open.
- Restore focus to triggering control on close.
