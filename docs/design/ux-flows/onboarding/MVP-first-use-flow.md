# MVP First-Use Flow

Status: Draft  
Version: MVP-001

## Goal

Guide a new user to first value without requiring education, registration, or setup complexity.

## Flow

1. User lands in the app.
2. User sees primary navigation for Tea Shelf, Teaware Shelf, and Brew Timer.
3. User adds first tea.
4. User adds first teaware item.
5. User starts first brew session.
6. User saves first session.

## Supported Alternate First-Use Paths

- User adds tea and starts a session without teaware.
- User skips catalog setup and starts a custom session.
- User adds tea only, then returns later to add teaware.
- User exits after creating inventory without starting a session.

## Requirements

- No account wall in MVP.
- No AI prompt in MVP.
- No academy or article gate in MVP.
- Empty states must support action, not explanation-heavy education.
- First-use guidance must not block users who already know what they want to brew.
- Beginner defaults may simplify timing setup, but must not introduce academy content.
