# MVP KPIs

Status: Approved for Implementation
Version: MVP-001

## Product Success Metrics

- Activation: user creates at least one tea, one teaware item, and one brew session.
- Inventory depth: average number of teas and teaware items per active user.
- Ceremony completion: percentage of started brew sessions that complete at least one steep.
- Repeat practice: percentage of users who repeat or reference a prior session.
- Stock maintenance: percentage of active teas with updated remaining quantity.

## Quality Metrics

- Core flow completion without errors.
- Responsive layout coverage across mobile, tablet, and desktop.
- Accessibility pass for keyboard navigation, focus states, contrast, and timer readability.

## Scope Health Metrics

- Count of MVP requirements mapped to Tea Shelf, Teaware Shelf, or Brew Timer.
- Count of deferred requests attempting to enter MVP.
- Approved change-control decisions affecting MVP scope.
