# Service Boundaries

Status: Approved for Implementation
Version: MVP-001

## MVP Boundary

Tea Shelf MVP contains these product domains:

- Tea Shelf
- Teaware Shelf
- Brew Timer
- Session History

The MVP has no required external services.

## Future Boundaries

- User System: registration, login, profile, language settings, and cloud ownership.
- Sync Service: multi-device persistence and conflict handling.
- Knowledge Shelf: articles, guides, taxonomy education, and content publishing.
- AI Assistant: recommendations, optimization, and personalized guidance.
- Commerce: monetization, affiliate stores, subscriptions, and marketplace functions.

## Boundary Rules

- MVP implementation must not require AI, content, commerce, or social services.
- Authentication and sync are MVP+ only unless separately approved.
- Future service boundaries must be captured in architecture docs before implementation.
