# Teaware List Row / Card

Status: Ready for Implementation Planning
Version: MVP-001

## Purpose

Summarize one teaware item.

## Layout

- Same base card layout as Tea List Row.
- Content grid: main identity, volume/material metadata, method chips, actions.
- Minimum row height: 88px.

## Content Order

1. Teaware name.
2. Category and material.
3. Volume.
4. Brewing method chips.
5. Archive/restore action.

## Rules

- Volume displays in `ml`.
- Archived teaware cannot be selected for new sessions by default.
