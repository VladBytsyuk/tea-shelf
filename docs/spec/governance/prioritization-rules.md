# Prioritization Rules

Status: Draft  
Version: MVP-001

## Rules

- MVP work must directly support tea inventory, teaware inventory, guided brewing, session history, archive/restore, or local data durability.
- MVP+ work is useful soon after MVP but not required for release, such as auth and cloud sync.
- Deferred work must not appear in MVP acceptance criteria.
- Repeat session is MVP Must.
- Recipe favorites and standalone recipe management are Deferred.
