# Value Proposition

Status: Draft  
Version: MVP-001

## Core Value

Tea Shelf helps users organize, improve, and deepen their personal tea practice.

## User Value

- Enthusiasts keep an accurate record of owned teas and remaining stock.
- Ceremony practitioners can run repeatable multi-steep sessions.
- Collectors can track teaware purpose, volume, material, and brewing compatibility.
- Beginners can start with structured inventory and brewing flows before advanced learning features exist.

## Commercial Positioning

Tea Shelf is positioned as a personal tea ceremony assistant for modern tea lovers. The MVP should prove daily and weekly utility before adding monetization.

Future commercial levers may include premium AI recommendations, advanced inventory analytics, tea aging tools, cloud backup, exclusive guides, affiliate tea stores, and marketplace features. These are explicitly out of MVP scope.

## Differentiation Assumptions

- Assumption: Existing general inventory tools do not capture tea-specific brewing and session context well.
  Impact if wrong: Positioning may need to shift toward design quality, ritual UX, or recommendation depth.

- Assumption: Early users value repeatable sessions and collection memory more than social discovery.
  Impact if wrong: Roadmap priority may shift, but MVP scope should remain protected until validated.
