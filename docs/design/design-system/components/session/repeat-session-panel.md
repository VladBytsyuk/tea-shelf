# Repeat Session Panel

Status: Ready for Implementation Planning
Version: MVP-001

## Purpose

Start a new timer setup from saved session snapshots.

## Layout

- Surface: `surface.default`.
- Border/radius: standard panel.
- Padding: 20px.
- Section gap: 16px.

## Rules

- Shows source session label and date.
- Allows sequence edit before starting.
- Does not overwrite original session.
- Does not require restoring archived source records.
