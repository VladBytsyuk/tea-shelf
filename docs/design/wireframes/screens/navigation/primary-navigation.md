# Primary Navigation Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Provide access to every primary MVP destination without exposing deferred modules.

## Layout

- Content: Tea Shelf, Teaware Shelf, Brew Timer, Session History.
- Mobile layout: compact top or bottom navigation with four reachable destinations, visible current destination, and no secondary module links.
- Tablet/desktop layout: horizontal or side navigation is acceptable if current page remains visible and content width is not squeezed below usable form/list widths.
- Accessibility: current item exposes current-page state; icon-only variants require accessible names.
