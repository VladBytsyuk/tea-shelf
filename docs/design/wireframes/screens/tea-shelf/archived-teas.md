# Archived Teas Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Restore archived teas without mixing them into active brewing selection.

## Layout

- Header: title "Archived Teas", back link to Tea Shelf.
- Body: archived tea rows/cards with archived badge, key metadata, restore action.
- Warning: archived teas are hidden from default active lists but historical session snapshots remain readable.
- Empty state: state that no archived teas exist and offer return to Tea Shelf.
