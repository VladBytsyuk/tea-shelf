# Steep Complete Layout

Status: Approved for Implementation
Version: MVP-001
Layout Version: LAYOUT-001

## Purpose

Show that the current steep is done and let the user advance intentionally.

## Layout

- Main region: "Steep complete" text, completed steep number, next steep preview or session completion message.
- Actions: "Next steep" when more steps exist; "Complete session" for final steep.
- Cancel remains available through confirmation.

## Accessibility

- State is visible as text and not conveyed by color or sound alone.
